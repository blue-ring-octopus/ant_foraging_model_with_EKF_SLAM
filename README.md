# Simple Ant Foraging Model

First time setup:
```
pip install mesa
pip install numpy
pip install math
```

To run the simulation without visuals:
```
cd Ant-Model
python run.py
```

To run the simulation with visuals:

```
cd Ant-Model
python server.py
```
go to http://127.0.0.1:8521 in the browser.
